ReadMe

Works in both Firefox and Chrome. Firefox supports both hardware accelerated rendering and software rendering. Chrome only seems to support hardware accelerated. Therefore, if you are using this in a Remote Desktop session, you might want to try to use Firefox if you want to use the 3D rendering.

-----

Right-click a node or edge to travese the graph 1 deep starting with that node. Check "Append Results" so that results are added to the graph.

-----

Click a node or edge to load its properties.

-----

Results are streamed from the server and can be cancelled.
